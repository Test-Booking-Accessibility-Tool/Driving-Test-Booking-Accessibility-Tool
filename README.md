## Installation Guide

### Step 1 – Download

Download the latest ZIP from:

**Releases → Latest Version**

### Step 2 – Extract

Unzip the folder.

### Step 3 – Install in Chrome

1. Open:

chrome://extensions

2. Enable:

**Developer Mode**

3. Click:

**Load unpacked**

4. Select:

**Test Booking Accessibility Tool**

---

## Pin the Extension (IMPORTANT)

1. Click the puzzle piece icon
2. Find Test Booking Accessibility Tool
3. Click the pin icon

---

## How to Use

1. Login to GOV.UK normally
2. Reach:

"Please choose one of the test centres below"

3. Open extension
4. Set date range
5. Select interval
6. Click:

**Start Watching**

## Security Checks
